# Plugin policy

Enable a plugin only when its capability justifies discovery context, tools, hooks, supply-chain risk and maintenance.

## Personal

Use productivity and learning plugins selectively.

## Work

Use only employer-approved plugins. External MCP services require privacy review.

## Phoxia

Plugins need an owner, version, changelog, evaluation and rollback.

## Before installing

Review origin, manifest, version, skills, agents, hooks, MCP servers, LSP servers, permissions, executed code, telemetry and update policy.

Do not version plugin caches or marketplace clones.

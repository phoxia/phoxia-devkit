# HTML, CSS and browser platform

- Use semantic HTML before ARIA.
- Preserve keyboard access, focus visibility and screen-reader labels.
- Avoid layout shifts and unnecessary client JavaScript.
- Measure Core Web Vitals and bundle impact.
- Use responsive design and robust text wrapping.
- Respect reduced motion and contrast.
- Treat URL, HTML and CSS injection as security boundaries.
- Test loading, empty, error and degraded states.

# Solidity

- Use the compiler and OpenZeppelin versions pinned by the project.
- Apply checks-effects-interactions and explicit access control.
- Consider reentrancy, oracle manipulation, front-running, rounding and upgrade storage layout.
- Minimize external calls and privileged authority.
- Use fuzzing, invariant tests and static analyzers.
- Treat gas optimization as secondary to correctness unless measured.
- Require independent audit before production value is at risk.

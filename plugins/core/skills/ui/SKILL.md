# UI workflow

Record reference hashes, viewports, fonts, tokens, dimensions, assets, interactions, states, responsiveness and accessibility.

Implement from the approved contract.

For visual tests, render baseline and current at identical viewports and produce a diff. Do not update a baseline only to make CI pass.

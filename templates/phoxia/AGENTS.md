# Phoxia Agent Constitution

Accepted RFCs, project metadata, architecture decisions, versioned contracts, implementation, tests and operations are the sources of truth.

- Products own their domains and private persistence.
- Cross-domain integration uses versioned APIs, commands or Pulse events.
- Projections are rebuildable.
- Commands request intent.
- Events record completed facts.
- Pulse consumers are idempotent.
- Replay is authorized, scoped, observable and audited.
- Security, privacy, accessibility, cost and observability are architecture requirements.
- Material features include data, contracts, failure analysis, tests, operations and release impact.

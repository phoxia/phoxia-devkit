# Visual fidelity

Store approved design references:

```text
design/reference.html
design/VISUAL-CONTRACT.md
design/screenshots/
design/assets/
design/tokens.json
```

Record viewports, fonts, spacing, dimensions, colors, assets, interactions, states, responsiveness and accessibility.

Visual regression means rendering the approved baseline and current implementation at the same viewport, then showing baseline, current and pixel diff.

It does not mean only displaying alternative designs before choosing a default.

For multiple Phoxia frontends, create a versioned design system named `phoxia-ui` with tokens, components, accessibility contracts, documentation and visual baselines.

# Naming

Monorepo applications:

```text
apps/web
apps/api
apps/mobile
apps/desktop
apps/worker
apps/cli
apps/docs
```

Shared packages:

```text
packages/ui
packages/contracts
packages/sdk
packages/config
packages/testing
```

Prefer `web` and `api` over generic `dashboard` and `server`.

Use `dashboard` only when it names a real product surface.

Use `Phoxia ‚ÄĘ Page` for page titles. Avoid em dashes in normal interface copy.

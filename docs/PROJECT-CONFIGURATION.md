# Project configuration

Project work is performed inside Codex or Claude, not through shell project subcommands.

```text
$phoxia-devkit project init
$phoxia-devkit project adopt
$phoxia-devkit project sync
$phoxia-devkit project audit
```

Claude uses `/phoxia-core:devkit`. The agent inspects existing instructions and `.phoxia/project.yaml`, preserves user-owned text, chooses proportional rigor and delegates to the installed project/documentation/legal skills. Use `project guide` when the next step is unclear.

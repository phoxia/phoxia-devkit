# Skills

Claude Code plugin commands retain their plugin namespace, such as `/phoxia-core:dev` and `/phoxia-open:phoxia`.

Repository and Codex adapters use portable names:

```text
$phoxia-dev
$phoxia-project
$phoxia-release
$phoxia-docs
$phoxia-ui
$phoxia-handoff
$phoxia-core-language
$phoxia-devkit
$phoxia-open
$phoxia-specialist
```

Use `$phoxia-devkit help` in Codex or `/phoxia-core:devkit help` in Claude for the contextual command list. `project guide` finds the next action; `mode direct` selects concise evidence-led responses. Commands remain English and responses follow the user's language.
